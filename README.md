## Client
#### Arduino
WIFI 모듈을 이용하여 자바와 소켓 통신, 가속도 센서를 이용하여 문의 움직임을 서버로 전달

## Server
#### Web Application Server
JSP를 통해 관리자가 문의 정보를 확인하고 제어
JDBC를 이용하여 아두이노가 보내온 정보들을 데이터베이스에 쓰고 읽는 역할 수행

#### Java Socket Server
아두이노와 소켓 통신하기 위한 서버. 신호 제어 및 센서 정보 등을 관리
